## Differentiator copy (use in all submissions)

When filling free-text fields, lead with what makes nostr-bray different from AbdelStark/nostr-mcp:

> Most Nostr MCP servers offer 5–15 tools for basic posting. nostr-bray provides 238 tools with trust-aware defaults: feeds filter out untrusted content automatically, DMs annotate sender trust scores, and reply tools warn about unknown authors. Unique capabilities include AI-to-AI dispatch over encrypted Nostr DMs, zero-knowledge age/income proofs, ring signatures for anonymous group membership, CANARY duress detection, and an L402 marketplace. Supports NIP-46 bunker auth so keys never leave a dedicated signing device.
